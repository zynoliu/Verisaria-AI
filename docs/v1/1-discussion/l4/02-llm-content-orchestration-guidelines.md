# Verisaria v1 LLM 内容编排功能指南（L4）

**状态**：初始草案（L4 Round 1–5 讨论后）
**日期**：2026-07-04
**版本**：v0.1

---

## 1. 功能定位

**半自动模式**为唯一支持的模式。

---

## 2. 禁止模式

- 纯辅助工具模式（LLM 只生成草稿，无验证）
- 接近全自动模式（LLM 生成 → 指标通过 → 可直接发布）

---

## 3. 强制流程

1. 生成后必须运行内容质量指标验证
2. 验证失败触发修复循环（最多 3 次）
3. 最终必须由人工确认后才能发布

---

## 4. 关键约束

- 无低风险豁免：所有 LLM 生成/修改的内容包都必须经过人工最终确认
- 审计日志：每次生成必须记录 prompt 版本、验证结果、修复次数、人工确认状态
- 高风险内容（faction-gated truth、复杂护送链、长链充分性）触发更严格验证
- 作者不能关闭 Must 指标的自动修复，但可选择“仅报告模式”
