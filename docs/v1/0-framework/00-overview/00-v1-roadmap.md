# Verisaria v1 轻量 Roadmap（v0.1）

**版本**：v0.1  
**日期**：2026-07-04  
**性质**：轻量、可迭代版本（vibe coding 环境适用）  
**更新机制**：每完成一个阶段或方向有重大调整时更新

---

## 1. v1 整体目标（简述）

构建一个**更大规模、更多涌现、可玩性导向**的 LLM 驱动 RPG 世界引擎，同时显著降低内容 authoring 成本。

核心支柱：
- 内容质量指标体系（Must 强制 + Recommended 分层）
- LLM 辅助内容编排（半自动 + 验证 + 人工确认）
- 分层真相世界模型（五层：Unknown / Canonical / Faction / Forbidden / Personal）
- 后果三通道 + FactLedger 强化
- GameSession 架构拆解 + 协议层可扩展设计

## 5. 关键风险与依赖

| 风险/依赖 | 说明 | 应对策略 |
|-----------|------|----------|
| **Authoring 成本失控** | 五层模型 + visibility 配置可能增加作者负担 | 工具支持 + 默认规则 + 验证器辅助 |
| **系统复杂度上升** | GameSession 拆解 + FactLedger 五层支持会增加复杂度 | 通过接缝良好封装，控制技术债 |
| **可玩性验证效果** | 内容质量指标是否真的有效，需要实际验证 | 优先做验证器原型，用真实内容包测试 |
| **LLM 生成质量** | 半自动模式的效果高度依赖模型能力和 prompt 工程 | 早期阶段多做小规模实验 |
