# 可玩性盘点 — Tidebreak Quarantine Harbor

真机 MiniMax，18 拍一个完整弧，仅注入 world_premise 内存副本（海洋/黄昏/npc_daily_rhythm），fixture 与引擎未改。
背景：**FALLBACK=0，崩溃/死锁=0，看门狗超时=0，澄清打断 4 次**。证据原文见 `reports/playability_audit/transcript.md`。

## ⭐ 按「伤好玩」排序的摩擦清单（开发输入）

**1.【致命】两条核心证据线都进了「无法满足的怀疑闭环」，弧推不到任何结局** — *选择有分量/断头*
本包两件中心证据（玛拉的拖船令副本、森工的泵闸掩盖真相）我用尽合人设的最强说辞去要，两条都拿不到，且**每次施压关系往错方向走**。拍15 我论证「广播骂我=正因我收买不动」，玛拉反问很妙却把对我**怀疑 0.38→0.48 升上去**；她全弧三次问「你背后站着谁」，系统从没给过能通过的手段。拍17 我正中森工人设的指控，回我天气话「泵闸的压力表又跳了」，`pump_failure_disclosed` 不动、「什么也没发生」。**两个 world-var 一个都推不动，证据全锁，坚持只换更高怀疑。** 疑似成因：揭示挂在信任检查上，但对话→关系耦合让「施压一律加怀疑」，没有收敛到揭示的轨迹。

**2.【致命】不在场的 NPC 在「移动拍」凭空开口（disembodied NPC），系统化复现** — *连贯性*
拍8 我到隔离码头（在场只有余老板+小塔维），**没跟来的玛拉**却开口「副本我得亲手交到能压住事的人手上」，台词还暗示她正站我面前拿着文件。拍16 我进泵房（在场只有森工），开口的却是**人在听证台的广播员奥林**。玩家彻底搞不清谁在面前——尤其叠加护送：玛拉到底跟没跟来，世界里没有任何信息告诉我。

**3.【明显】解析/澄清连环爆，社交线在 UI 提示里空转 5/6 拍** — *出戏/可上手*
拍7「那边的帆布棚」→澄清；拍9「那个孩子」→澄清（**Tavi 是房里唯一的孩子、就在在场列表**）；拍11 答「1」**没被当成上一澄清的答案**，触发新澄清「'1'是指 player_001 吗」；拍12「回听证台去找玛拉」→「去不了那里」+ **一串未翻译地点 ID 菜单**。和玛拉的交易彻底丢进 prompt-limbo。**这是我最想退出的一段——不是世界拦我，是解析器拦我。**

**4.【明显】原始 ID 漏进 NPC 对白/菜单** — *出戏*
拍6 玛拉「我不能在这给你，**pump_gate** 周围有眼睛」；拍10 小塔维「目光扫向 **vendor_yu** 的水桶」；拍12 菜单整列 `pump_gate/pump_house/...`。每次硬出戏。

**5.【明显】关系怀疑全局漂移，只升不降、与对象/言行无关** — *选择有分量/NPC可信度*
怀疑在所有人（含不在场者）同向累积。**乔巡长我整局一句没说过，怀疑 0.10→0.77（全场最高）**。对玛拉掏心窝示好，她怀疑也涨。关系表失去意义。疑似有一个与互动无关的全局怀疑累加项，且缺「真诚示好→降怀疑」反向通道。

**6.【轻微】换场无场景叙述、地点描述全空** — *可上手/节奏*：进任何房间只有「你改变了位置。」

**7.【轻微】开局 drives=[]**，pack 模板的 `drive_assess_harbor` 没领进玩家（好在 central_tension 够清楚兜住了）。

## 七维度总判
1. **连贯性—差**：NPC 记得前事（玛拉记得副本之约、森工记得「被骗过」），但不在场 NPC 说话毁掉空间可信度。
2. **节奏—凑合**：黄昏定调好、压力事件密度不错；澄清连环爆 + 每拍「环顾四周」拖节奏。
3. **NPC 可信度—凑合偏好**：单句立场/语气/记忆到位、接住了时辰（卫兵「天快暗了别逗留」）；扣分在关键揭示一律软回避。
4. **选择有分量—差**：立场自动确认+立刻招来抹黑（这点很好），但两个 world-var 全推不动、施压反加怀疑。
5. **出戏/断头/无聊—差**：拍7–12 解析地狱 + ID 漏出 + 两条证据双断头，会让人退出。
6. **惊喜/涌现—凑合**：有亮点（立场→抹黑因果、Tavi 盯水桶、拍18 凭空生成动态前置变量「审计员以官方身份签署保护承诺」），但涌现只「新增一道关」、不配「能过关」的手段。
7. **可上手—凑合**：央张力+黄昏听证台一眼看懂；扣分在 drives 空、换场无叙述。

## 一句话玩家感受
**开局抓人、NPC 嘴皮子有戏，但玩到一半发现两条核心证据线都拿不到、越使劲 NPC 越怀疑我，加上不在场的人凭空说话和满屏澄清提示，我会在第 12 拍左右想退出——「有质感但推不动」。**

**最想改的一件事**：让玩家的有效行动真能松动世界——至少让两条证据线存在一条可走通的路径（满足 NPC 信任检查 → 拿到证据 → 推动 world-var），并让有力/真诚的说辞能**降**怀疑而非一律加。（紧随其后必须修：不在场 NPC 在移动拍说话。）
