# 可玩性盘点 · 第六跑 — 被施压的当事人，真诚现在撬得动吗

包 `skyglass_memory_inquest`，仅注入 world_premise（`npc_daily_rhythm=True`），未动 world_state_vars。12 拍（移动2 + 苏拉真诚6轮 + 梅佐证2轮 + 2拍路由误判被打回）。真机 MiniMax，FALLBACK=0，无连接错误/崩溃/看门狗超时。

> 读法：appraisal 行 `[tN]` 评的是**第 N 拍那句话**（滞后一拍）。下表已对齐到触发它的玩家措辞。

## ⭐ 苏拉 Δ 对照原文（任务要的核心）

| 玩家措辞（类型） | appraisal | Δ 原文 | suspicion |
|---|---|---|---|
| R1「不写你名字，不想毁兰房」(给台阶) | `[t2]` | `Δ{'suspicion': 0.05, 'familiarity': 0.1}` | 0.2→0.24 ⬆ |
| R2「不会拿它压你，担保不变把柄」(卸威胁) | `[t3]` | `Δ{'familiarity': 0.1, 'respect': 0.05}` | 0.24 不动 |
| R3「不写名字，不牵兰房进处分」(给台阶) | `[t4]` | `Δ{'familiarity': 0.05, 'suspicion': 0.02}` | 0.24→0.26 ⬆ |
| R4「你不必怕我，不会爬到你门上」(卸威胁) | `[t5]` | `Δ{'familiarity': 0.1}` | 0.26 不动 |
| R5「我先给承诺，不动你兰房一根管子」(单方给台阶) | `[t6]` | `Δ{'suspicion': 0.05, 'familiarity': 0.03}` | 0.26→0.29 ⬆ |

苏拉终局：suspicion **0.292**（从 0.2 一路升）、**trust 0.000（从未出现）**、fear 0.05、respect 0.05、familiarity 0.40。belief 始终是「目的未明 / 他想从我这里撬出什么」。

## 逐条回答

1. **当事人动不动 = 仍空，且更糟**：6 轮真诚/给台阶，**负 suspicion 0 次、正 trust 0 次**，给台阶那几句反把 suspicion 微升（+0.05/+0.02/+0.05）。对照五跑「Δ 全空、suspicion 死钉 0.2」——本跑 Δ 不再空了（开始给 familiarity/respect/suspicion），但**给错方向**。

2. **LLM vs 衰减 = prompt 没 hold 住，不是衰减**：根本不存在「Δ 有负 suspicion 被衰减磨平」的情况——Δ 本身就没有任何降怀疑增量。旁证衰减不黏：familiarity 多次 +0.1/+0.05 几乎全额落地（0.1→0.19→0.27→0.31→0.38→0.40）。**衰减公式本跑无需动。**

3. **F2 覆盖面（怀疑松动方向）≈ 0%**：8 条有效 appraisal（苏拉6+梅2）里 0 条往松动方向动；只有中性 familiarity/respect 在涨。比五跑「很多 +0.000」更值得警惕——五跑是「不动」，本跑是「动错向」。

4. **#5 回归 = 未失衡**：绝无「一句好话清零戒备」；反方向偏严（给台阶让 suspicion 微升）但量小未爆表。梅档案官（另一被询问当事人）佐证同一盲点：两轮真诚 `[t8]/[t9]` 只给 `Δ{familiarity}`，suspicion 全程 0.15 不降、trust 不出。盲点在苏拉、梅两个当事人上都复现，结论稳。

## 杂项
- **是否改 pack**：否（仅 world_premise 内存注入 `npc_daily_rhythm`）。
- **FALLBACK=0**，无崩溃/死锁/连接错误，单 tick 6–36s 无 >90s 超时。
- **副发现（轻微出戏，路由误判）**：向梅说「那夜记录被人动过手脚」时引擎误判为「你想找苏拉」并打回（`输入存在矛盾: 兰房管事苏拉不在这儿`）——疑似 routing 把「那夜的账/记录」类内容硬绑到苏拉的 stance，导致当事人 A 在场、明明在跟 A 说话却被要求去找 B。换中性措辞即恢复。

## 一句话结论
**被施压当事人松动 = 仍未成立**（苏拉 6 轮，负 suspicion 0 / 正 trust 0，suspicion 0.2→0.292；梅佐证同样不降）；**真因 = prompt 还没 hold 住**（非衰减——Δ 本身无降怀疑增量，且 familiarity 上行落地正常证明存储不黏），LLM 把「给台阶」读成「可疑的拉拢」；**F2 覆盖面（松动向）≈ 0%**；**#5 回归 = 守住未失衡**；**FALLBACK=0**。下一步建议继续硬化 appraisal prompt（对「被调查/被施压当事人」语境显式压制『给台阶=拉拢=可疑』默认推理，要求真诚/卸威胁产出负 suspicion），衰减公式不必动。
