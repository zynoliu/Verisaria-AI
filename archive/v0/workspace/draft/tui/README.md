# Verisaria TUI Draft

Standalone Textual prototype with mock data only. It does not import or modify
anything under `src/`.

Run from the project root:

```bash
.venv/bin/python draft/tui/verisaria_tui_mock.py
```

Keys:

- `m`: toggle the map panel
- `f`: cycle the focus panel
- `w`: flip the mocked world variable
- `q`: quit
